# 同一电脑上使用两个GitHub账号

需要进行以下几个步骤：
- 生成公钥
- 新建config文件并编辑
- 在对应文件夹下设置

全局可以不用取消，用的比较多的那个账户就设置为全局；
用的少的那个就在对应的文件夹下设置。
